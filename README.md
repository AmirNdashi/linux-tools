# Linux Tools

### sysinfo.sh
A simple Linux system information tool built using Bash.

## Features
- Shows system uptime
- Displays memory & disk usage
- Lists top 5 CPU-consuming processes
- Shows IP address

## Usage
1. Make script executable:
   chmod +x sysinfo.sh

2. Run:
   ./sysinfo.sh

## Installation
sudo mv sysinfo.sh /usr/local/bin/sysinfo

Then run:
sysinfo
